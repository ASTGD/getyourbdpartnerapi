GetYour Partner API — Domain Order

Base URL: https://getyour.com.bd

Open Swagger UI

Authentication

HTTP Basic Auth. Use your User ID as username and password as password.

Endpoint

POST /api/v1/domain/orders

Content-Type: multipart/form-data • Rate limit: 60/min

Request fields

  • domain (string) — required
  • nameServers[0], nameServers[1] (string) — required
  • nameServers[2] (string) — optional
  • fullName (string) — required
  • nid (10/13/17 digits) — required
  • nid_document (jpg/jpeg/png/pdf) — required
  • registration_document (file) — required for some TLDs
  • email (email) — required
  • contactAddress (string) — required
  • contactNumber (+880 + 10 digits) — required
  • years (1–10) — optional

cURL example

curl -X POST "https://getyour.com.bd/api/v1/domain/orders" \
  -u "USER_ID:PASSWORD" \
  -H "Accept: application/json" \
  -F "domain=example.com" \
  -F "nameServers[0]=ns1.example.net" \
  -F "nameServers[1]=ns2.example.net" \
  -F "fullName=Minhazul Islam" \
  -F "nid=1234567890" \
  -F "nid_document=@/path/to/nid.pdf" \
  -F "registration_document=@/path/to/reg.pdf" \
  -F "email=minhajulshaoun@gmail.com" \
  -F "contactAddress=Dhaka, Bangladesh" \
  -F "contactNumber=+8801712345678" \
  -F "years=1"

Errors

  • 401 Unauthorized (Basic Auth failed)
  • 422 Validation error (Laravel standard JSON)
  • 429 Too Many Requests (throttled)
